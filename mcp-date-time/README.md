# MCP Date-Time Server

MCP-сервер для получения текущей даты и времени через SSE.

## Описание

Сервер предоставляет инструменты для работы с датой и временем:
- Получение текущей даты и времени
- Получение текущей даты
- Получение текущего времени
- Список поддерживаемых временных зон
- Конвертация даты и времени между зонами

## Использование

SSE endpoint: `http://localhost:8003/sse`

### Доступные инструменты

1. **get_current_datetime** – текущие дата и время
2. **get_current_date** – текущая дата
3. **get_current_time** – текущее время
4. **list_timezones** – список временных зон
5. **convert_datetime** – конвертация между зонами

## Запуск

```bash
docker-compose up -d mcp-date-time
```
