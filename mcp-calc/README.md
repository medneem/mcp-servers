# MCP Calculator Server

MCP сервер для выполнения математических вычислений, полезный для LLM.

## Возможности

- **Базовые арифметические операции**: сложение, вычитание, умножение, деление, возведение в степень, корень.
- **Научные вычисления**: тригонометрия, логарифмы, факториал, округление.
- **Работа с выражениями**: вычисление математических выражений в строковом формате.
- **Статистика**: среднее, медиана, стандартное отклонение.
- **Матричные операции** (с использованием numpy): умножение матриц, определитель, обратная матрица.
- **Символьные вычисления** (с использованием sympy): упрощение выражений, решение уравнений, дифференцирование.

## Инструменты (Tools)

### `calculate_expression`
Вычисляет математическое выражение, заданное строкой.

**Параметры:**
- `expression` (строка): математическое выражение, например `"2 + 3 * 4"`.
- `precision` (целое число, опционально): количество знаков после запятой для округления результата (по умолчанию 10).

**Возвращает:** числовой результат в виде строки.

### `basic_arithmetic`
Выполняет базовую арифметическую операцию между двумя числами.

**Параметры:**
- `operation` (строка): одна из `"add"`, `"subtract"`, `"multiply"`, `"divide"`, `"power"`, `"root"`.
- `a` (число): первое число.
- `b` (число): второе число (для `"root"` — степень корня).

**Возвращает:** результат операции.

### `scientific_function`
Применяет научную функцию к числу.

**Параметры:**
- `function` (строка): `"sin"`, `"cos"`, `"tan"`, `"log"` (натуральный логарифм), `"log10"`, `"exp"`, `"sqrt"`, `"factorial"`, `"abs"`.
- `x` (число): входное значение.
- `angle_unit` (строка, опционально): `"degrees"` или `"radians"` для тригонометрических функций (по умолчанию `"radians"`).

**Возвращает:** результат функции.

### `solve_equation`
Решает алгебраическое уравнение символьно.

**Параметры:**
- `equation` (строка): уравнение в виде строки, например `"x**2 - 4 = 0"`.
- `variable` (строка, опционально): переменная для решения (по умолчанию `"x"`).

**Возвращает:** список решений.

### `matrix_operation`
Выполняет операцию над матрицами.

**Параметры:**
- `operation` (строка): `"multiply"`, `"determinant"`, `"inverse"`, `"transpose"`.
- `matrix_a` (список списков чисел): первая матрица.
- `matrix_b` (список списков чисел, опционально): вторая матрица (для умножения).

**Возвращает:** результирующую матрицу или число.

### `statistical_summary`
Вычисляет статистические показатели для набора чисел.

**Параметры:**
- `numbers` (список чисел): данные для анализа.
- `metrics` (список строк, опционально): список метрик для вычисления: `"mean"`, `"median"`, `"std"`, `"variance"`, `"min"`, `"max"`, `"sum"` (по умолчанию все).

**Возвращает:** словарь с вычисленными метриками.

## Запуск

### Локально (с использованием uv)

```bash
cd mcp-calc
uv sync
uv run python server.py
```

Сервер будет доступен на `http://localhost:8005/sse`.

### Через Docker

```bash
docker build -t mcp-calc .
docker run -p 8005:8005 mcp-calc
```

### В составе docker-compose

Добавьте сервис в `docker-compose.yml` (см. корневую директорию проекта).

## Интеграция с MCP клиентом

Настройте клиент (например, Claude Desktop) для подключения к SSE эндпоинту `http://localhost:8005/sse`.
